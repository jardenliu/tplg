# 模板处理工具

本工具用于将模板文件中的占位符替换为对应的值，目前支持的占位符替换，具体如下：

| 占位符               | 说明                                 |
| -------------------- | ------------------------------------ |
| `{{name}}`         | 原始名称参数                         |
| `{{pascalName}}`   | 帕斯卡命名法格式（大驼峰）           |
| `{{currentDate}}`  | 当前日期                             |
| `{{version}}`      | 项目版本号                           |
| `{{username}}`     | 当前用户名                           |
| `{{email}}`        | 用户邮箱                             |
| `{{snakeName}}`    | 蛇形命名法格式                       |
| `{{kebabName}}`    | 烤肉串命名法格式（短横线连接）       |
| `{{constantName}}` | 常量命名法格式（全大写，下划线连接） |
